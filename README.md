# Heart-Disease-Prediction-using-ML-algorithms

Algorithms used:
  - Logistic regression
  - Random forest classifier
  - K-Nearest neighbour
  - XGBoost Classifier	

Dataset:
  - The Dataset is from UCI Machine Learning Repository, In this there are 76 attributes from which 14 are being used in this model.
  - 14 attributes are Age, Gender, Chest pain type, Cholesterol, and some ECG values.
  - Dataset consists of 303 instances

Modules:
  - Data Collection 
  - Data Cleaning
    - pandas library
  - Data Preprocessing
    - Data is well cleaned and manipulated and then the data is split for evaluation process
  Data Visualization
    - Matplotlib, Seaborn libraries
